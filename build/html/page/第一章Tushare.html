第一章 Tushare¶

这一章介绍的是量化交易中经常用到的数据接口——Tushare，Tushare免费提供股票数据，目前提供了四种获取数据的方式，分别为 http, Python SDK, Matlab SDK, R SDK。 本章以Python语言为例，说明如何从tushare获取特定股票的信息。

本章将从什么是Tushare、Tushare的导入、基本的语句等方面入手，手把手地教你对Tushare入门，开启量化交易的第一步——交易数据获取。
